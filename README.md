# kisuke.vim

## My Vim Split Pane Sandbox

After around ~1 year and ~260 commits on this project I really felt bad about the initial presentation of it. This project is a ultra primitive LLM chat integration with local session persistance + file marking for next prompt context but thats all. I revisited to play with mcp client + server implementation but the moment I achieved the most basic stdio connection work with tool approvals I realized the current state of opencode infra and existance of ACP. There is less than 0 value this plugin offers when you can spin up anything else in tmux split instead of Vim split.

[Old README.md](https://github.com/dorukozerr/kisuke.vim/blob/6d97b3ef1d5c5b0c04f4b3afff0decd242707cf6/README.md) with installation and usage.
